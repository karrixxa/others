Current goal

Current issues

Symmetry breaking
- Two neurons often end up with nearly identical weights.
- Competition never resolves into a single winner.
- Cannot rely on lucky random initialization.
- Possible solutions:
  * Euclidean distance / geometric influence
  * Stronger inhibitory learning
  * Topology changes
  * Learning algorithm changes

Weight saturation
- Some neurons hit the weight cap.
- Competition stalls.
- Investigate weight updates, learning rates, and saturation.

Inhibitory learning
- Based on accumulated charge.
- Runner-up neurons should receive stronger inhibition.
- Consider increasing inhibitory learning rate.

- Winning neuron fires more frequently over time.
