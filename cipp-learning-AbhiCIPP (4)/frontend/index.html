Signed input
+1 active · −1 inactive

Signed-spike learning: on each competitor fire, its active pixels (+1) potentiate and its inactive pixels (−1) depress — dw = η·p·(1−(w/w_cap)²)·signal. Each 3×3 grid is one competitor's feedforward receptive field, mapped by each sensory afferent's input pixel; a ring marks the current winner and units that can no longer reach threshold are flagged dead. Cells with no feedforward edge for that pixel are blank.
