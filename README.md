# 🏛️ Hermes AI Evolution: Technical Manifesto & Project Vault

This repository serves as the definitive archive of the experimental journey from simple sandbox games to high-performance computational engines and autonomous AI orchestration.

## 🎯 The Narrative: A Calibration Journey
This program was not a collection of random scripts, but a structured **Calibration Journey**. We used increasingly complex domains to prove the AI's ability to move from **Autonomous Action** $\rightarrow$ **Pattern Recognition** $\rightarrow$ **State Tracking** $\rightarrow$ **Technical Performance**.

---

## 📂 Architectural Index

### 01. High-Performance Compute (`/01_high_perf_compute`)
**The "Technical Ceiling" Challenge.**
- **Project:** Hyper-Pi Engine.
- **Goal:** Calculate millions of digits of $\pi$ with maximum throughput and minimum energy.
- **Elite Engineering:** 
    - Implemented the **Chudnovsky Algorithm** with **Binary Splitting**.
    - Leveraged **GMP (GNU Multi-Precision)** for FFT-based multiplication.
    - Scaled across 192 CPU cores using a **Logarithmic Tree Merge** to eliminate bottlenecks.
- **Result:** Achieved $\sim 184.5$ Million digits/minute in Python.

### 02. Reinforcement Learning (`/02_reinforcement_learning/pong`)
**The "Action-Response" Calibration.**
- **Project:** Pong AI.
- **Evolution:**
    - **PPO/DQN:** Basic agent training from scratch.
    - **Imitation Learning:** Training the AI to mimic expert human play.
    - **Physics Optimization:** Moved from frame-based movement to **$\Delta t$ (Delta-Time) scaling** for consistent physics across varying hardware.

### 03. Pattern Recognition (`/03_pattern_recognition/mnist`)
**The "Perception" Calibration.**
- **Project:** MNIST Digit Detection.
- **Approach:** Comparison between traditional **CNNs** and modern **Vision Transformers (ViT)**.
- **Outcome:** Validated the ability to deploy high-accuracy models into lightweight FastAPI applications.

### 04. Complex State Logic (`/04_complex_state_logic/dnd`)
**The "Cognitive" Calibration.**
- **Project:** Hermes RPG / DnD System.
- **Challenge:** Moving from simple "if/then" logic to managing a complex, persistent world state.
- **Engineering:** Developed a specialized API to bridge LLM reasoning with a structured game database.
- **Knowledge Base:** This project is supported by a vast library of architectural skills in `/knowledge_base/skills/`, covering **Distributed Backend Orchestration**, **C2 Gateway Integration**, and **Tactical Agent Intelligence**.


### 05. Project OS (`/05_project_os`)
**The "Infrastructure" Layer.**
- **Mattermost Sentinel:** A closed-loop autonomous monitor (Detect $\rightarrow$ Reason $\rightarrow$ Respond) running every 60s.
- **Research Portal:** A rootless FastAPI + SQLite hub designed to bypass strict cluster restrictions.
- **Taxonomy:** Implementation of the `[RESEARCH] \rightarrow [EXECUTION] \rightarrow [VALIDATION]` loop.

## 📉 The "Complexity Wall" & Final Pivot
As the system evolved, we encountered a critical architectural realization: **the gap between Digital Logic and Analog-style Brain computation.**

While we mastered the "Digital" path (high precision, deterministic algorithms, $O(n \log n)$ complexity), the project ultimately pivoted toward understanding **Neuromorphic Computing**. We realized that as the creating system grew more complex, the "Python Tax" and the rigidity of Turing-Machine-style algorithms became bottlenecks. This led to our final research into **Free Energy Minimization** and **Causal Learning**, seeking a more "biological" approach to AI.
